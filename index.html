Criador de Tarefas

Um projeto de criador de tarefas usando HTML, CSS e JavaScript para praticar a lógica de programação em JavaScript. Criei este projeto em curso de JavaScript na Udemy e fazendo esse projeto aprendi novos conceitos, uma coisa legal é que quando você sai do site suas tarefas são salvas pois elas ficam são salvas no LocalStorage do navegador que é uma mini base de dados do navegador. Eu também aprendi a como controlar as teclas do teclado, pois nesse exercício quando o usuário digitar a tarefa dele e clicar no botão Enter, o botão vai obedecer ao comando então a palavra que a gente usa é keyCode com o evento keypress e nós manipulamos a tecla pelo seu número, no caso do Enter é o 13. Eu também aprendi a converter um elemento JavaScript para uma string no formato JSON e aprendi a converter um valor de volta para um array, quando fazemos isso nós estamos convertendo de volta para um objeto JavaScript.

Tecnologias: HTML, CSS e JS.
